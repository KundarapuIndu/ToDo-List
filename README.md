# ✅ Todo App

A simple and clean **Todo List** web app built using HTML, CSS, and JavaScript.

## ✨ Features
- Add and delete tasks
- Mark tasks as completed
- Clean and responsive UI

## 💻 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
